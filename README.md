# TaxPal-A-Personal-Finance-and-Tax-Estimator-for-Freelancers
Developed a full-stack personal finance dashboard using React, Express, and MongoDB for real-time income, expense, and budget tracking. Added dynamic CRUD budgets, ₹ display, Recharts visuals, tax estimation with PDF/CSV reports, and secure JWT authentication with modular, reusable UI components.
• Developed a full-stack Personal finance dashboard using React.js, Express.js and MongoDB Atlas, enabling users to track income, expenses and budgets in real time.  
• Implemented dynamic budget management with CRUD operations, live currency display (₹) and automatic “spent” value updates in the database.  
• Designed an interactive dashboard with visual insights using Recharts, featuring expense breakdowns, recent transactions and tax summaries. 
• Integrated tax estimation and report generation (PDF/CSV formats) for monthly and annual summaries to assist freelancers in financial planning. 
• Built secure user authentication with JWT and modular, reusable UI components (sidebar, topbar, settings panel) for maintainable scalability.
